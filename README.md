> *"A matemática desenvolve o cálculo. A lógica transforma isso em solução."*

Este repositório reúne exercícios desenvolvidos em **JavaScript** com foco em cálculos matemáticos e resolução lógica de problemas envolvendo áreas geométricas.


O objetivo principal é fortalecer a lógica de programação, interpretação matemática e construção de soluções utilizando JavaScript.

---

#  Sobre os Exercícios

Todos os exercícios seguem uma estrutura organizada contendo:

-  Entrada de valores
-  Declaração de variáveis
-  Aplicação de fórmulas matemáticas
-  Exibição dos resultados
-  Resolução lógica estruturada

Cada algoritmo foi desenvolvido para melhorar o raciocínio computacional através de problemas matemáticos práticos.

---

- Entrada e saída de dados
